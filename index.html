Creating and building with a custom operator
Introduction

This project is a live, working demo of an app running on RedHat OpenShift, built on a custom operator.

We developed this project as a reference example, and a set of resources, for other teams to build their own operators, and applications that use them.

The app itself is a flight booking tool, using Janus Graph as a database. The custom operator is an operator for Janus Graph.
